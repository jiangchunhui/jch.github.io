MySQL 1. 基礎SQL語句

查看数据库和表：

SHOW DATABASES; --顯示数据库 USE test; --選擇数据库 SHOW TABLES; --顯示表 DESC t_table_name; --顯示表結構

CREATE DATABASE MyDatabase; -- 创建数据库 CREATE DATABASE MyDatabase DEFAULT CHARSET utf8; -- 创建数据库，使用默认字符utf-8 关系数据库的一個重要概念:任何數據都表示為行和列組成的表，而每條 SELECT語句的結果也都是一個行和列組成的表。

複制表：

CREATE TABLE test2 SELECT * FROM test1; --複製表包括數據 CREATE TABLE test2 SELECT * FROM test1 WHERE 0; --複製表結構但不要數據 INSERT INTO test2 SELECT * FROM test1; --表結構相同的時候複製數據 INSERT INTO person_copy(id,name) SELECT id,name FROM person; --僅複製部分列，可以加上where語句複製部分行 批量錄入數據，語法：

LOAD DATA [LOCAL] INFILE 'file_name.txt' [REPLACE | IGNORE] INTO TABLE tbl_name 例子：

LOAD DATA INFILE 'data.txt' INTO TABLE tbl_name FIELDS TERMINATED BY ',' [OPTIONALLY] ENCLOSED BY '"' LINES TERMINATED BY '\n'; 導出數據，使用shell：

echo "select * from test_table" | mysql -h127.0.0.1 -P3306 -uroot -p123456 test -A > data.txt --mysql加上-N配置可以設置不要輸出表頭 常用MySQL函數：

SELECT COUNT(*) FROM dual; --計數 SELECT curdate(); SELECT curtime(); SELECT now(); SELECT rand(); 執行外部sql文件：

SOURCE test.SQL; . test.SQL; --或者這樣也可以 權限：

GRANT priv_type (COLUMNS) ON what TO USER IDENTIFIED BY “password” WITH GRANT OPTION 管理服務器：

SHOW processlist; --顯示當前連接到服務器的鏈接 2. 常用客戶端

很好用的工具是SQLyog，也是收費軟件，也有破解版。它最大的特色是提供可編輯的SQL文本框，可以選擇部分SQL語句執行，按F8快捷鍵就執行，非常方便。

常用的软件还有Navicat，這是收費軟件，有很多破解版綠色版下載。它支持MySQL、Oracle、SQLite、PostgreSQL等。一般使用它來瀏覽數據塊和表，很方便地創建表、修改表，支持流量表數據和修改，命令行也很方便。
